# Security Policy

## Supported Versions

Only the latest published `dart-decimate` version receives security fixes.

## Reporting a Vulnerability

Please report security issues through GitHub private vulnerability reporting for
`sgaabdu4/dart-decimate`.

Do not open a public issue for secrets, supply-chain compromise, code execution,
credential exposure, or other sensitive reports.

## Scope

Security review covers:

- Rust CLI and MCP server behavior
- npm wrapper and install scripts
- GitHub Actions release and publish workflow
- dependency and supply-chain hygiene

`dart-decimate security` reports local review candidates in Dart/Flutter
repositories. Those findings are static-analysis prompts, not confirmed
vulnerabilities.
